# Contributing

WICID is open source under the MIT License. You're welcome to fork the project and make it your own.

**Pull requests are not actively reviewed.** If you'd like to extend or modify WICID, the best path is to fork the repository and build on it independently.

## Getting Started

1. Fork the repository
2. Set up your development environment: [`docs/DEVELOPER_SETUP.md`](docs/DEVELOPER_SETUP.md)
3. Run the quality checks before committing:
   ```bash
   pipenv run pre-commit run --all-files
   ```
