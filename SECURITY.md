# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in WICID firmware, please report it responsibly.

**Do not open a public issue for security vulnerabilities.**

Instead, please email: **security@wicid.ai**

Include:
- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Suggested fix (if any)

## Scope

This policy covers the WICID firmware source code in this repository. It does not cover:
- Third-party CircuitPython libraries in `src/lib/`
- The CircuitPython runtime itself
- Hardware vulnerabilities

## Response

We will acknowledge receipt within a reasonable timeframe and work to address confirmed vulnerabilities. As this is a community project with limited bandwidth, response times may vary.
